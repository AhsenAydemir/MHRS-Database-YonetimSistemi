# MHRS-Database-YonetimSistemi
PostgreSQL ile geliştirilmiş Hastane Randevu Sistemi (MHRS) veritabanı tasarımı ve SQL sorguları.
### Öne Çıkan Teknik Özellikler
Veritabanı Yönetimi: PostgreSQL kullanılarak ilişkisel veritabanı tasarımı yapılmıştır.
Gelişmiş Mantıksal Yapılar: Veri tutarlılığı ve otomasyon için Trigger, Stored Procedure ve Transaction yapıları uygulanmıştır.
Performans Optimizasyonu: Sorgu hızlarını artırmak amacıyla stratejik noktalar için Index'ler tanımlanmıştır.
Modelleme: Proje; Kavramsal ER Diyagramı, Güncel ER Modeli ve Class Diyagramı dökümantasyonlarını içermektedir.
